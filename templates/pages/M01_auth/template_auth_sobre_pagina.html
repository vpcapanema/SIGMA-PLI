Sistema de Gerenciamento Integrado de Módulos Administrativos

O que é o SIGMA-PLI?

O SIGMA-PLI é o Módulo de Gerenciamento de Cadastros do Sistema SIGMA (Sistema de Gerenciamento Integrado de Módulos Administrativos), desenvolvido para centralizar e facilitar o cadastro de pessoas físicas, pessoas jurídicas e usuários do sistema.

Principais Funcionalidades

  • Cadastro de Pessoas Físicas com validação de CPF
  • Cadastro de Pessoas Jurídicas com validação de CNPJ
  • Gestão de Usuários do Sistema
  • Controle de Solicitações de Cadastro
  • Gerenciamento de Sessões e Segurança
  • Interface Responsiva e Intuitiva

Segurança e Privacidade

O sistema utiliza as melhores práticas de segurança, incluindo criptografia de senhas, autenticação de usuários, controle de sessões e proteção contra ataques comuns.

Tecnologias Utilizadas

Backend: FastAPI, Python 3.13+
Frontend: Bootstrap 5, Jinja2
Banco de Dados: PostgreSQL, Neo4j
Autenticação: JWT, Sessions
