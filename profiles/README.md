# Perfis do VS Code (SIGMA)

Estes arquivos `.code-profile` podem ser importados em: Gerenciar (ícone de engrenagem) → Perfis → Importar Perfil… → Arquivo.

Arquivos:
- `SIGMA-Core.code-profile`
- `SIGMA-Dados.code-profile`
- `SIGMA-DevOps.code-profile`

Após importar, os perfis aparecerão no menu Gerenciar → Perfis.

Observação: o conteúdo inicial é mínimo (sem settings/extensões). Você pode personalizar cada perfil após importar.
